# Smart Expense Tracker

An **interactive Python tool** to track and analyze personal expenses. Users can add, remove, and categorize expenses, and get **category-wise totals** through a **clean console interface**.

This project demonstrates **modular coding, object-oriented programming, and professional coding practices** in Python.

## Features
- Interactive console menu for adding/removing expenses
- Category-wise totals summary
- Modular and reusable Python code using classes and functions
- Error handling for invalid input
- Expandable: optional visualizations with Matplotlib/Seaborn
  
## Tech Stack
- Python
- Matplotlib / Seaborn (for visualization)

## How it Works
- Run `main.py` to start the tracker
- Follow the interactive menu:
    - Add expense: Enter amount and category
    - Remove expense: Remove by index
    - Show all expenses
    - Show category-wise summary
    - Exit
- The tracker displays clean, readable outputs in the console
